# 🛒 Blinkit Grocery Sales Data Analysis

Welcome to the Blinkit Data Analysis project! This is a **real-world Business Intelligence case study** that leverages **SQL, Python, Excel, and Power BI** to uncover valuable insights from Blinkit's grocery sales and outlet data. From sales patterns to customer ratings and outlet efficiency, this project demonstrates an end-to-end data analytics workflow.

---

## 📌 Objective

The primary objective of this project is to analyze Blinkit's sales data to identify:

- 🔍 Customer purchasing patterns
- 📉 Underperforming products and outlets
- 📈 Revenue growth opportunities
- 📊 Actionable business insights for decision-making

This project simulates how a Data Analyst transforms raw business data into meaningful insights for stakeholders.

# 📊 Business Questions & KPI Analysis

The dashboard answers the following business questions:

### 🔸 Total Sales by Fat Content

- Compare Low Fat vs Regular products
- KPIs: Total Sales, Average Sales, Number of Items, Average Rating
- Visualization: Donut Chart

### 🔸 Total Sales by Item Type

- Identify the highest revenue-generating product categories
- Visualization: Bar Chart

### 🔸 Fat Content Sales Across Outlet Types

- Compare sales contribution of different fat categories across outlet types
- Visualization: Stacked Column Chart

### 🔸 Sales by Outlet Establishment Year

- Analyze how outlet age impacts sales performance
- Visualization: Line Chart

### 🔸 Sales by Outlet Size

- Determine which outlet sizes contribute the highest revenue
- Visualization: Donut Chart

### 🔸 Sales by Outlet Location

- Analyze revenue distribution across different outlet locations
- Visualization: Funnel Map

### 🔸 KPI Comparison by Outlet Type

- Compare Total Sales, Average Sales, Ratings, and Number of Items
- Visualization: Matrix Card

# 📌 Interactive Power BI Dashboard

The Power BI dashboard is **fully dynamic and interactive**, allowing users to explore the data effortlessly.

### Dashboard Features

- ✅ Dynamic slicers and filters
- ✅ Switch between different outlet types and item categories
- ✅ Interactive cross-filtering between visuals
- ✅ Drill-down capability for detailed analysis
- ✅ Dynamic KPI cards that update instantly based on user selections
- ✅ Professional dashboard layout for business presentations

Instead of displaying static charts, every visual responds dynamically to user interactions, making it easy to analyze different business scenarios in real time.
